Gida Azkarra / Quick Guide

# StockBaseApp - Instalazio Gida Azkarra

Dokumentu honek aplikazioa modu egokian instalatzeko eta abiarazteko urratsak azaltzen ditu.

Instalazio Urratsak
  1. Deskarga: Sartu administratzaile gisa webgunean, joan goiko eskuineko "Instaladorea" botoira eta egin klik "Deskargatu instaladorea" aukeran.
  2. Segurtasuna (Garrantzitsua): Fitxategia deskargatu ondoren, joan .zip fitxategira, egin klik eskuineko botoiarekin eta hautatu Propietateak. "Orokorra" fitxan, markatu "Desblokeatu" laukia eta sakatu Ados.
  3. Deskonprimitu: Atera ZIP fitxategiko edukia nahi duzun karpeta batean. Gomendagarria da "Deskonprimitu StockBaseApp/" aukera erabiltzea.
  4. Exekuzioa: Sartu karpetan, ondoren “win-x64” karpetan eta egin klik bikoitza StockBaseApp.exe fitxategian.
Ohar Garrantzitsuak
  • Sare Konexioa: Ziurtatu zure gailua "praktikak" sarera konektatuta dagoela. Pasahitza: izarpark.
  • Konfigurazioa: Ez da aparteko konfiguraziorik behar.

# StockBaseApp - Quick Installation Guide

This document explains the steps to correctly install and launch the application.

Installation Steps
  1. Download: Log in as an administrator, go to the top-right "Instaladorea" button and click "Deskargatu instaladorea".
  2. Security (Important): After downloading, right-click the .zip file, select Properties, check "Unblock" and click OK.
  3. Unzip: Extract the ZIP contents into your desired folder. Recommended: "Extract to StockBaseApp/".
  4. Execution: Open the folder, go into “win-x64” and double-click StockBaseApp.exe.
Important Notes
  • Network Connection: You must be connected to the "praktikak" network. Password: izarpark.
  • Configuration: No additional setup is required.
